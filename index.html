Travel Planner (Breadth First Search and Dijkstra’s Algorithm) (Java)

Applying Dijkstra’s algorithm and breadth first search, this collaborative project can parse an input CSV and return a list of transport options that is either the most direct (fewest stops), cheapest, or fastest in total minutes! Travel planner demonstrates strong comprehension of model view controller interactions in object oriented programming.

Travel Planner


Decision Trees (Java)

Using basic machine learning principles, this decision tree project can process input files and predict a target attribute with a minimum accuracy of 70%. This decision tree splits on random nodes using a random number generator and builds a tree with default attributes at each value in case of an erroneous or unfamiliar input.

Decision Tree


Maze Game (Pyret)

Using functional programming in Python derivative Pyret, I constructed a maze game that you can play! This program processes a google spreadsheet and maps out a list of images into a maze, a player, and a set of objects that you can interact with!
